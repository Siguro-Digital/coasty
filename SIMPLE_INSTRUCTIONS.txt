           COASTY AUTOMATION - SUPER SIMPLE GUIDE (Mac)
═══════════════════════════════════════════════════════════════

                    ⭐ JUST DOUBLE-CLICK! ⭐

═══════════════════════════════════════════════════════════════

STEP 1: INSTALL NODE.JS (Do this once)
   └─→ Go to: https://nodejs.org/
   └─→ Click "Download" (big green button)
   └─→ Install it
   └─→ Restart your computer

═══════════════════════════════════════════════════════════════

STEP 2: SETUP (Do this once)
   └─→ RIGHT-CLICK:  Setup.command
   └─→ Select "Open"
   └─→ If you see "Apple cannot verify", click "Open" again
   └─→ Wait 2-5 minutes
   └─→ Done! ✅

═══════════════════════════════════════════════════════════════

STEP 3: RUN THE PROGRAM (Every time you want to use it)
   └─→ RIGHT-CLICK:  Run.command
   └─→ Select "Open"
   └─→ If you see "Apple cannot verify", click "Open" again
   └─→ Follow the menu on screen

═══════════════════════════════════════════════════════════════

IMPORTANT: If you see "Apple cannot verify this app"
   └─→ Right-click the file (don't double-click)
   └─→ Select "Open"
   └─→ Click "Open" when macOS asks again
   └─→ This is normal - macOS is protecting you!

If right-clicking doesn't work, see QUICK_START.md for help.
